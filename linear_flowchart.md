## Detailed Flow Steps

### **Phase 1: Entry & Input (Top)**
1. **Email Input** â†’ User enters email address
2. **Field Selection** â†’ Choose generation or group type
3. **Input Validation** â†’ System validates and parses input (supports: 2027, 27, g27, g2027, gen27, gen2027)

### **Phase 2: Page Transition (Upper Middle)**
4. **Header Display** â†’ Shows generation/group title
5. **Gallery Loading** â†’ Fetches photos from server
6. **Counter Setup** â†’ Initializes photo counter

### **Phase 3: User Interaction (Middle)**
7. **Photo Click** â†’ User selects individual photos
8. **Selection Toggle** â†’ Photos are marked as selected
9. **Visual Feedback** â†’ Selected photos get grey overlay

### **Phase 4: Actions (Lower Middle)**
10. **Select All** â†’ Bulk selection/deselection
11. **Send Selection** â†’ Submit selected photos
12. **Back Navigation** â†’ Return to previous page

### **Phase 5: Processing (Bottom)**
13. **Validation Check** â†’ Ensure photos are selected
14. **Server Request** â†’ Send data to backend
15. **Success Message** â†’ Confirm completion
